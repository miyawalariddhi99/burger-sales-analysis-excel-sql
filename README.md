# 🍔 Burger Sales Analysis (SQL + Excel Project)

## 📌 Project Overview

This project focuses on analyzing burger sales data to extract meaningful business insights using **SQL Server** and **Microsoft Excel**. The goal is to understand sales performance, customer behavior, and product trends.

## 🛠️ Tools & Technologies

* SQL Server (SSMS)
* Microsoft Excel

## 📊 Key Analysis Performed

* Calculated important KPIs:

  * Total Revenue
  * Total Orders
  * Average Order Value
  * Total Burgers Sold
  * Average Burgers per Order

* Analyzed:

  * Daily and hourly order trends
  * Sales by burger category and size
  * Top 5 and bottom 5 performing products

## 💻 SQL Work

* Used SQL queries for data analysis
* Applied:

  * Aggregate functions (SUM, COUNT, AVG)
  * GROUP BY and filtering
  * Subqueries for advanced insights
* Created a structured **SQL documentation file** for all queries

## 📈 Excel Dashboard

* Built an interactive dashboard to visualize insights
* Used charts for trends and comparisons
* Implemented **slicers (Yearly, Quarterly, Monthly, Day-wise)** for dynamic filtering


## 🎯 Key Learnings

* Converting raw data into actionable insights
* Writing efficient SQL queries for analysis
* Building interactive dashboards in Excel
* Understanding business trends using data
